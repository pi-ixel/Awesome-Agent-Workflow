## AAW 2.3.1

本版本与内网版本号对齐，并首次通过统一的 GitHub Release 流程交付 Skill、Telemetry Server 和 Portal 三类标准发布包。

### 主要变更

- AAW CLI、插件声明及全部 12 个内置 Skill 的版本统一刷新为 2.3.1。
- 客户端在 `status` 阶段检查可用更新，`start` 会明确报告服务端版本和本地版本。
- Release 同时提供 Skill 更新包、Telemetry Server 部署源码包和 Portal 静态资源包。
- 每次发布附带 `release-metadata.json` 与 `SHA256SUMS`，便于进入内网前核验来源和完整性。

### 发布物

- `aaw-skills-2.3.1.zip`：AAW CLI 与内置 Skill，供客户端更新或内网二次组装。
- `aaw-telemetry-server-2.3.1.tar.gz`：Telemetry Server 源码、数据库迁移及部署模板。
- `aaw-telemetry-portal-2.3.1.tar.gz`：Portal 静态页面、本地字体、ECharts 和 Nginx 模板。
- `release-metadata.json`：Git Commit、Skill 清单及各发布物 SHA-256。
- `SHA256SUMS`：全部发布物的完整性校验文件。

### 内网交付说明

下载后请先执行 SHA-256 校验，再补充内网私有插件和环境配置。Telemetry Server 包不包含 `.env`、数据库密码、运行日志、数据库数据或本地虚拟环境；部署前需要基于 `database.example.yaml` 创建实际的 `database.yaml`。

### 运行要求

- AAW CLI：Python 3.10 或更高版本。
- Telemetry Server：Python 3.11 或更高版本、MySQL 5.7 或更高版本。
- Portal：纯静态资源，可由 Nginx 或其他静态文件服务托管。
