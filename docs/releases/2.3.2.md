## AAW 2.3.2

本版本统一发布 AAW CLI、全部 13 个内置 Skill、Telemetry Server 和 Portal，重点补齐设计门禁、遥测联调、任务执行与问题反馈能力。

### 主要变更

- 新增 `sr-design-gate` Skill，并在 AAW 工作流中加入 SR 设计门禁步骤。
- 全部 13 个内置 Skill、AAW CLI、Claude/Codex 插件声明、Telemetry Server 和 Portal 统一升级为 2.3.2。
- 优化 `task-split` 与 `task-dev` 的任务结构、执行约束和遥测适配。
- CLI 增强遥测上报，并统一网络访问策略与错误处理。
- Portal 新增问题与愿望反馈页面，Telemetry Server 提供配套接口和数据库迁移。
- 修订工作流说明、Skill 扩展说明和遥测接口契约。

### 发布物

- `aaw-skills-2.3.2.zip`：AAW CLI 与全部 13 个内置 Skill，供客户端更新或内网二次组装。
- `aaw-telemetry-server-2.3.2.tar.gz`：Telemetry Server 源码、数据库迁移及部署模板。
- `aaw-telemetry-portal-2.3.2.tar.gz`：Portal 静态资源及部署模板。
- `release-metadata.json`：Git Commit、Skill 清单及发布物元数据。
- `SHA256SUMS`：全部发布物的 SHA-256 校验值。

### 内网交付说明

下载后请先校验 `SHA256SUMS`，再补充内网私有插件和环境配置。发布包不包含数据库密码、运行日志、数据库数据或本地虚拟环境。

### 运行要求

- AAW CLI：Python 3.10 或更高版本。
- Telemetry Server：Python 3.11 或更高版本、MySQL 5.7 或更高版本。
- Portal：纯静态资源，可由 Nginx 或其他静态文件服务托管。
